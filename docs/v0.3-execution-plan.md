# v0.3 Execution Plan: Unified Developer Experience

v0.3 delivers a friendlier API surface over the verified v0.1 ZDD and v0.2 BDD foundations.

Execution evidence is tracked in `docs/backlog.md` under "v0.3 Execution Plan".

## Scope

v0.3 must include:

- `DecisionDiagramManager`
- BDD operator overloads for `!`, `&`, `|`, and `^`
- handle-first diagnostics extension methods
- high-level Markdown export extension methods
- NuGet package metadata
- README updates
- documentation skeleton for getting started and high-level APIs

## Quality Gate

v0.3 completion requires the normal project Definition of Done plus:

- 100% method coverage for changed production code
- method-level coverage review to confirm no changed function remains at 0%
- meaningful unit tests with an English purpose comment at the beginning of each new test method
- Arrange / Act / Assert structure for new unit tests
- tests that assert observable behavior rather than merely executing functions

## Out of Scope

- MTBDD/ZMTBDD implementation
- CodeAnalysis implementation
- CLI command parsing
- Graphviz process execution
- MDD / ADD work

## Verification Commands

```text
dotnet build DecisionDiagramSharp.slnx -v minimal
dotnet test DecisionDiagramSharp.slnx -v minimal
dotnet test tests/DecisionDiagramSharp.Core.Tests/DecisionDiagramSharp.Core.Tests.csproj -v minimal /p:CollectCoverage=true /p:CoverletOutputFormat=opencover /p:CoverletOutput=TestResults/core-coverage.xml
dotnet test tests/DecisionDiagramSharp.Diagnostics.Tests/DecisionDiagramSharp.Diagnostics.Tests.csproj -v minimal /p:CollectCoverage=true /p:CoverletOutputFormat=opencover /p:CoverletOutput=TestResults/diagnostics-coverage.xml
dotnet test tests/DecisionDiagramSharp.Export.Tests/DecisionDiagramSharp.Export.Tests.csproj -v minimal /p:CollectCoverage=true /p:CoverletOutputFormat=opencover /p:CoverletOutput=TestResults/export-coverage.xml
```

Coverage reports must be reviewed at method granularity.
